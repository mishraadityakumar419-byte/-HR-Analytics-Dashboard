📊 Power BI Dashboard Projects
This repository contains two interactive data analytics dashboards built using Power BI:
👨‍💼 Human Resources Analytics Dashboard
🎓 Student Performance Analytics Dashboard
These projects focus on extracting meaningful insights from datasets using visualization and KPI tracking.



📌 1. Human Resources Analytics Dashboard

🔍 Overview
This dashboard analyzes employee data to identify patterns in attrition, performance, job satisfaction, and workforce distribution.
📊 Key KPIs
Total Employees: 1.47K
Active Employees: 1.233K
Attrition Rate: 16.12%
📈 Insights Visualized
Employees by Performance Rating
Attrition by Department
Attrition by Job Satisfaction
Employees Left by Years at Company
Income vs Experience Analysis
Work-Life Balance Impact
🎯 Objective
To help HR teams:
Reduce employee attrition
Improve employee satisfaction
Make data-driven workforce decisions


📌 2. Student Performance Analytics Dashboard


🔍 Overview
This dashboard analyzes student data to understand factors affecting academic performance.
📊 Key KPIs
Pass Percentage: 85.10%
Average Absences: 4.01
Average Final Grade: 12.14
📈 Insights Visualized
Performance by School
Impact of Family Support (famsup)
Performance by Gender
Study Time vs Grades
Impact of Internet Access
Absences vs Study Time
🎯 Objective
To help educators:
Identify weak students early
Improve teaching strategies
Understand key performance factors
🛠️ Tools & Technologies
Power BI (Data Visualization)
Excel / CSV Dataset
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)
